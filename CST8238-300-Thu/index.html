Course Overview

Course Details

  • Course Code: CST8238
  • Prerequisites: CST8108, CST8116, and CST8215
  • Hours: 70 normative hours
  • Format: 5 hours online per week + 2 hours self-study

Important Note

For this course, you will be required to bring your own mobile computing device (laptop) that will enable you to download the appropriate free software. PC or Mac systems are both acceptable.

What You'll Learn

HTML5 & CSS3

Master modern web markup and styling techniques including CSS Grid and Flexbox with Bootstrap 4.

JavaScript

Learn client-side scripting and front-end frameworks like Angular for interactive web applications.

PHP Development

Implement server-side scripting with PHP, including object-oriented programming and database integration.

AJAX & JSON

Create dynamic Web 2.0 applications with asynchronous data exchange and modern JavaScript libraries.
