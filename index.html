CST8238 Web Programming - Midterm Answer Key

Section A: Internet and Web Fundamentals

1. Default HTTP Port

Answer: C) 80
  • Port 80 is the default port for HTTP traffic
  • Port 443 is used for HTTPS (secure connections)
  • Port 21 is for FTP (File Transfer Protocol)
  • Port 8080 is commonly used as an alternative HTTP port

2. DNS Function

Answer: A) Domain Name System - maps domain names to IP addresses
  • DNS is a fundamental system that translates human-readable domain names into IP addresses
  • Example: www.google.com â†’ 172.217.xxx.xxx
  • Without DNS, we would need to remember IP addresses for every website

3. Domain Name Components

Answer: C) Top-level domain

In www.algonquincollege.com:

  • .com is the top-level domain (TLD)
  • algonquincollege is the second-level domain
  • www is a subdomain

4. GET vs POST

Answer: C) GET is visible in the address bar, POST data is hidden 
// GET example - Data visible in URL
http://example.com/search?query=test&page=1

// POST example - Data hidden in request body
Content-Type: application/x-www-form-urlencoded
query=test&page=1

Section C: CSS Fundamentals

13. Bold Text Syntax

Answer: A) p {font-weight: bold;} 
/* Correct way to make all paragraphs bold */
p {
    font-weight: bold;
}

/* Alternative using font-weight value */
p {
    font-weight: 700;
}

14. CSS Specificity

Answer: C) ID selector (e.g., #myId)

Specificity hierarchy (highest to lowest):

  1. Inline styles
  2. ID selectors (#myId)
  3. Class selectors (.myClass)
  4. Element selectors (p, div)
  5. Universal selector (*)

15. CSS Descendant Selector

Answer: C) Selects all <p> elements that are inside <div> elements at any level 
/* Matches any p inside any div */
div p {
    color: red;
}

/* vs. direct child selector */
div > p {
    color: blue;
}

16. CSS Box Model

Answer: B) padding

Box Model from inside out:

  1. Content
  2. Padding (inside spacing)
  3. Border
  4. Margin (outside spacing)

17. Margin vs Padding

Answer: B) Margin adds space outside elements, padding adds space inside 
.box {
    /* Space inside the element */
    padding: 20px;

    /* Space outside the element */
    margin: 20px;
}

18. Background Color

Answer: B) background-color 
.element {
    background-color: #ff0000;  /* Red */
    /* Or */
    background-color: rgb(255, 0, 0);
    /* Or */
    background-color: red;
}

19. Box Model Calculation

Answer: C) Total width = 250px 
.box {
    width: 200px;        /* Content width */
    padding: 20px;       /* 20px on each side = +40px */
    border: 5px solid;   /* 5px on each side = +10px */
}
/* Total: 200 + 40 + 10 = 250px */

20. Hover Pseudo-class

Answer: B) :hover 
/* Changes color when mouse hovers over element */

    background-color: blue;
    color: white;
}

21. Flexbox Container

Answer: B) Creates a flexible box layout container 
.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

23. String Concatenation

Answer: B) "55" 
console.log(5 + "5");  // "55" (string concatenation)

24. Equality Comparison

Answer: C) true, false 
let x = 5;    // number
let y = "5";  // string

console.log(x == y);   // true (loose equality, type coercion)
console.log(x === y);  // false (strict equality, no type coercion)

25. Array Method

Answer: B) push()

26. Array Length

Answer: B) 3 
const arr = [1, 2, 3];
console.log(arr.length);  // 3 (counts elements in array)

27. Arrow Function

Answer: B) const square = (x) => x * x 
// Arrow function syntax
const square = (x) => x * x;

}

28. This Keyword

Answer: B) Refers to the current execution context 
class Example {
    constructor() {
        this.value = 42;  // 'this' refers to instance
    }

    method() {
        // 'this' refers to instance in method
        console.log(this.value);
    }
}

29. Hoisting

Answer: B) undefined 
function test() {
    // var a is hoisted but not initialized
    console.log(a);  // undefined
    var a = 10;      // actual declaration
}

// Equivalent to:
function test() {
    var a;           // hoisted declaration
    console.log(a);  // undefined
    a = 10;         // initialization
}

30. Array Iteration

Answer: C) for (let item of arr) 
const arr = [1, 2, 3];

// Modern way (best)
for (let item of arr) {
    console.log(item);
}

// Also good for arrays
arr.forEach(item => console.log(item));

// Traditional way (less preferred)
for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}

31. JSON Methods

Answer: B) Converts a JSON string to a JavaScript object 
// String to Object
const jsonString = '{"name": "John", "age": 30}';
const obj = JSON.parse(jsonString);
console.log(obj.name);  // "John"

// Object to String
const objToConvert = {name: "John", age: 30};
const jsonStr = JSON.stringify(objToConvert);
console.log(jsonStr);  // '{"name":"John","age":30}'

32. Callback Function

Answer: B) A function passed as an argument to another function 
// Example of callback function
function doSomething(callback) {
    console.log("Doing something...");
    callback();
}

doSomething(() => {
    console.log("Callback executed!");
});

33. Closure Example

Answer: C) 1, 2, 3 
function makeCounter() {
    let count = 0;          // Private variable
    return function() {     // Closure
        count++;            // Accesses parent scope
        return count;
    };
}

const counter = makeCounter();
console.log(counter());  // 1
console.log(counter());  // 2
console.log(counter());  // 3
